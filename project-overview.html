Asset Closet

Enterprise IT Asset Inventory Management System for Austin and SCV TCS-Apple Operations

Version 2.0 — Next.js Migration

📋 About the Project

Asset Closet is an internal IT asset tracking system built for the Austin and SCV TCS-Apple storage closet teams. It manages the complete lifecycle of IT equipment — laptops, monitors, keyboards, and other peripherals — from check-in to check-out, with full audit trails and leadership reporting.

Key Users: Storage closet operators (iPad), team leads (MacBook), and leadership (reports & analytics).
Deployment: Runs on a local network, accessible via any device on the same Wi-Fi.

🔄 What Changed in Version 2.0

The application was migrated from Python Streamlit to Next.js (React) for better performance, modern UI, and improved security.

Previous Version (v1.0)

  • Built with Python Streamlit
  • Single-page app with session state
  • CSV file storage
  • Basic styling with custom CSS
  • No security headers or rate limiting

New Version (v2.0)

  • Built with Next.js + React + TypeScript
  • Multi-page app with JWT authentication
  • Excel workbook storage (auto-migrated from CSV)
  • Apple-standard UI with dark mode
  • Security headers, rate limiting, input sanitization

Key Improvements

  • 3x faster page loads — React renders instantly vs Streamlit's full page reloads
  • Works offline — once loaded, the app works without constant server communication
  • Better iPad experience — touch-optimized UI, floating labels, PIN keypad
  • Security hardened — JWT sessions, encrypted cookies, CSP headers, rate limiting
  • Dark mode — full theme toggle across all pages
  • Session management — idle timeout with warning, refresh logout detection
  • Workbook caching — the app trims Excel's inflated Austin sheet range and caches parsed rows in memory for sub-second reloads

📥 Check-In

Register new assets or return existing ones with full metadata.

  • 12 data fields: Asset ID, Serial Number, Chip, Configuration, Processor, Model Year, RAM, Assigned To, Location, Emp ID, Email, Status
  • Auto-fill from existing records when Asset ID or Serial Number is entered
  • Camera OCR scan to read asset stickers (02HW0...) and engraved serial numbers
  • Asset ID validation (must start with 02HW0, 01HW0, 34HW0, 3HW0, or 4HW0)
  • Auto-navigates to Check-Out after successful check-in

📤 Check-Out

Assign available assets to users.

  • Dropdown showing available assets (excludes Special Case and already checked-out)
  • Asset details auto-displayed on selection
  • Camera OCR scan to quickly select assets
  • Required: Assigned To field

📦 Inventory

View, search, and manage all assets.

  • Site-aware inventory with Austin and SCV sheet selection
  • Split-panel table with pinned Asset ID column
  • Search with instant filtering and clear button
  • Rows per page selector (10, 25, 50, 100, 200)
  • Inline edit via floating-label modal
  • In-memory client cache for fast location revisits
  • Settings gear: Hide Unavailable, Fullscreen, All Columns, Export Excel
  • Low inventory warning when unallocated assets ≤ 10

👥 User Audit

Complete login activity tracking.

  • Tracks: Login Time, Logout Time, Duration, Logout Method, Device
  • Color-coded failure rows (red for incorrect PIN, invalid ID)
  • Logout reasons: User Logout, Session Timeout, Page Refresh
  • Search, pagination, Excel export

💻 Asset Log

Full history of every asset action.

  • Every check-in, check-out, and edit recorded with timestamp and actor
  • Search, pagination, Excel export

📊 Leadership Reports & Statistics

Real-time analytics dashboard for leadership visibility into asset utilization and team activity.

Location Add-on: Reports now default to Austin and can switch to SCV from a site dropdown.
Workbook counts: Austin = 1064 raw rows, SCV = 547 raw rows.
Performance: site payloads are cached in memory so revisits and site switches load in under a few seconds.

🎨 Theming & Design

Design Principles

  • Apple Human Interface Guidelines — floating labels, rounded corners, SF typography
  • WCAG AA+ Accessible — high contrast, keyboard navigation, screen reader support
  • Color-blind safe — errors use icon + text + border, not color alone
  • iPad-first — touch-optimized controls, PIN keypad, camera scan

📷 Camera OCR Scanning

Built-in camera scanning to quickly identify assets without manual typing.

What It Reads

  • White sticker Asset IDs: 02HW0, 01HW0, 34HW0, 3HW0, 4HW0
  • Engraved Serial Numbers: Text after "Serial Number" / "S/N" / "SN"

How It Works

  • Opens device camera (rear-facing)
  • Captures frame, enhances contrast
  • OCR via Tesseract.js (runs in browser)
  • Auto-fills form fields with results
  • Photo upload fallback for pre-taken images

🛡️ Security

Authentication

  • JWT tokens with 15-min expiry
  • httpOnly, sameSite cookies
  • Encrypted PIN storage (AES-128-CBC)
  • Server-side session validation

Protection

  • Rate limiting (10 login / 5 PIN per minute)
  • Content Security Policy headers
  • Input sanitization on all APIs
  • X-Frame-Options: DENY (anti-clickjacking)

⏱️ Session Management

  • Idle Timeout — 10 minutes of inactivity triggers a 5-minute countdown warning
  • Warning Popup — "Session Expiring Soon" with countdown and Extend/Logout buttons
  • Auto Logout — Force logout after timeout, recorded as "Session Timeout" in audit
  • Refresh Detection — Page refresh triggers logout, recorded as "Page Refresh" in audit
  • All session events logged in User Audit with timestamp and device type
